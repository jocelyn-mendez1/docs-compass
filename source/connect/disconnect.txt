=====================
Disconnect Connection
=====================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Disconnecting from a MongoDB instance closes the Compass connection to
the active instance and returns the Compass view to the initial
connection dialog.

Disconnect Procedure
--------------------

.. procedure::

   .. step:: Click :guilabel:`Connect` in the menu bar.

   .. step:: Select :guilabel:`Disconnect` from the dropdown menu.

      .. figure:: /images/compass/disconnect.png
         :figwidth: 500px
         :alt: Disconnect menu item

.. seealso::

   To create a new connection, see :ref:`connect-run-compass`.
